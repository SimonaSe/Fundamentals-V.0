Activité 2 - Choisir le niveau d’unité d’organisation et ajuster les options du tableau

Dans TRAININGLAND, ajouter à votre tableau les districts ayant déclaré des décès maternels. À noter que ces unités d’organisation peuvent différer de ce qui est montré dans la vidéo. 

Dans l’onglet Unité d’organisation :

  • Choisissez ‘District’ comme niveau d’unité d’organisation.

  • Ouvrez l’onglet Mise en page et :
      • Déplacez les unités d’organisation vers le bas dans Rangée
      • Déplacez les périodes vers la colonne

  • Ouvrez l’onglet Options et cochez ‘Masquer les rangées vides’.

  • Cliquez sur ‘Actualiser’ pour voir votre tableau.

Activité 3 - Enregistrer le tableau comme favori

Dans TRAININGLAND, ajoutez les établissements et enregistrez le tableau dans vos favoris.

Dans l’onglet Unité d’organisation :

  • Choisissez ‘Région Animal’ en tant qu’unité d’organisation
  • Choisissez ‘Établissement’ comme niveau

  • Cliquez sur ‘Actualiser’ pour voir votre tableau.

À noter qu’en réalité, il n’est pas forcément nécessaire d'indiquer votre nom d’utilisateur lorsque vous enregistrez un favori. Dans le cadre de cette formation, cela est toutefois utile pour identifier les favoris que vous avez créés.

Activité 4 - Vérifier les indicateurs et appliquer les blocs de légende et les libellés de tableau

Dans TRAININGLAND, créez un tableau contenant 2 indicateurs et appliquez une légende à ce tableau.

Dans l’onglet Données, sélectionnez :

  • ‘Indicateurs’ comme type de données.
  • ‘Vaccination - couvertures’ comme Groupe d'indicateurs
  • ‘Couverture BCG’ et ‘Couverture DTP3’ en tant qu’indicateurs.

Dans l’onglet Période :

  • Choisissez ‘6 derniers trimestres’ comme période. Les 6 derniers trimestres affichés peuvent être différents de ceux visibles dans la vidéo.

Dans l’onglet Unité d’organisation :

  • Modifiez le mode de sélection sur ‘Choisir les niveaux’.
  • Choisissez ‘District’ comme niveau d’unité d’organisation.

  • Modifiez la présentation pour reproduire ce qui est montré dans la vidéo. Vous pouvez actualiser le tableau si vous le souhaitez, mais il reste plusieurs étapes à terminer dans cette activité.

  • Ouvrez les options du tableau et modifiez les éléments suivants :
      • Supprimez tous les totaux du tableau.
      • Appliquer le bloc de légende Couverture EPI Vous pouvez essayer les options ‘Couleur du texte’ et ‘Couleur du fond’ du style de légende pour voir la différence.
      • Donnez un titre à votre tableau.

  • Cliquez sur Actualiser lorsque toutes les modifications ont été effectuées.

Le format du tableau doit ressembler à celui montré dans la vidéo.

Activité 1 - Ressources et rapports d’ensemble de données

Dans TRAININGLAND, veuillez exécuter un rapport d’ensemble de données. Pour accéder aux rapports d'ensemble de données :

  • Allez sur Applis -> Rapports -> Rapport d’ensemble de données

  • Sélectionnez les critères à appliquer au rapport d’ensemble de données et cliquez sur ‘voir le rapport’ lorsque vous avez terminé. Vous pouvez utiliser les éléments suivants pour vous guider :
    • Ensemble de données : Soins de santé de base - mensuel
    • Type de période : Annuelle
    • Période : 2016
    • Unité d’organisation : Région Animal (vous pouvez indiquer n’importe quelle unité d’organisation ici)

  • Vous accéderez aux résultats correspondants à la combinaison ensemble de données/période/unité d’organisation après avoir cliqué sur ‘voir le rapport’.

Comme nous l’avons mentionné dans la vidéo, l’ensemble de données vous est présenté comme il apparaitrait dans le formulaire de saisie. Toutefois, vous ne pouvez pas modifier les données ici ; souvent, cette vue est utile pour vérifier les données dans un format familier pour les utilisateurs. Les données peuvent être agrégées au sein du rapport, et donc analysées à différents niveaux de la hiérarchie en fonction des besoins.

  • Accédez aux ressources grâce au sous-menu situé à gauche.

  • Vous pouvez consulter une ressource en cliquant sur l’élément correspondant avec le bouton gauche de la souris et en sélectionnant ‘consulter cette ressource’.

Activité 2 - Rapports des unités d’organisation et résumé du taux de déclaration

Rapport des unités d’organisation

Lorsque vous êtes dans TRAININGLAND, vous pouvez accéder aux rapports des unités d’organisation :

  • Soit en les sélectionnant à partir du menu situé sur le côté dans l'appli Rapports,

  • soit en vous rendant sur Applis -> Rapports -> Rapport de répartition des unités d’organisation.

Pour lancer le rapport :

  • Définissez d’abord les critères que vous souhaitez examiner. Dans cet exemple, nous pouvons utiliser :
      • Rapport d’unité d’organisation : Région Alimentation
      • Ensemble de groupe : Type
  • Sélectionnez ‘Créer le rapport’ pour lancer la création du rapport d’unité d’organisation.

Le rapport sera exécuté et classera les établissements en fonction de leur type.

Résumé du taux de déclaration

Nous pouvons ensuite générer un résumé du taux de déclaration. Nous pouvons y accéder :

  • Soit en le sélectionnant à partir du menu situé sur le côté dans l'appli Rapports,

  • soit en nous rendant sur Applis -> Rapports -> Résumé du taux de déclaration.

  • Sélectionnez les critères à analyser dans le rapport. Dans cet exemple, nous pouvons utiliser :
      • Unité d’organisation : Région Alimentation
      • Ensemble de données : Soins de santé de base - mensuel
      • Type de période : Par trimestre
      • Période : octobre-décembre 2016
  • Ne décochez pas la case ‘Sur la base des ensembles de données complets’. Ceci signifie que le taux de complétude est calculé en fonction des utilisateurs ayant soumis un ensemble de données complet dans l'appli de saisie des données. Nous y reviendrons dans les sessions ultérieures.

  • Cliquez sur ‘Créer le rapport’ pour exécuter le rapport et voir les résultats.
